<h1 align="center">Kiwi - QR Code Generator</h1>

<p align="center">
  <b>A minimal, modern, and professional QR code generator with a premium AMOLED dark theme.</b>
</p>

---

## ✨ Features

- 🌑 **AMOLED Dark Theme**: Pure black background for a sleek, modern, and battery-saving experience.
- 📱 **Minimal UI**: Clean and professional design focused on ease of use.
- ⚡ **Instant Generation**: Converts any link or text into a high-quality QR code instantly.
- 💾 **Save to Gallery**: Download your generated QR codes directly to your device.
- 📤 **Fast Sharing**: Share your QR codes via other apps with just one click.
- 🚀 **Built with Jetpack Compose**: Modern Android development practices for performance and stability.

## 🛠️ Tech Stack

- **Kotlin**: The primary programming language.
- **Jetpack Compose**: Modern toolkit for building native UI.
- **ZXing**: Industry-standard library for QR code generation.
- **Material 3**: Google's latest design system for Android.

## 📦 Getting Started

To get a local copy up and running, follow these simple steps:

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/shejanahmmed/Kiwi-QR-Code-Generator.git
    ```
2.  **Open in Android Studio**: Open the cloned project folder in Android Studio Ladybug or later.
3.  **Build and Run**: Connect your device or emulator and click the "Run" button.
